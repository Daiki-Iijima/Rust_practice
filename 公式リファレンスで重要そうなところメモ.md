# ソースファイル名の命名規則
ファイル名に２単語以上使っている場合、「アンダースコア」(_)で区切る
## 例
- 悪
    ```
    helloworld.rs
    ```
- 良
    ```
    hello_world.rs
    ```

# インデントはタブではなく、スペース４個
インデントをつけるときはタブではなく、スペースを4個で
```rs
fn main {
    println!("hello,world!");
}
```
